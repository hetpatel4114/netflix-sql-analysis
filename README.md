# 🎬 Netflix Data Analysis Using SQL

## 📌 Project Overview
Analyzed Netflix's content library using MySQL to answer 13 business questions
around content type, ratings, countries, directors, and trends.

## 🗃️ Dataset
- Source: Kaggle / Netflix Titles Dataset
- Records: ~8,800 rows | 12 columns

## 🛠️ Tools Used
- MySQL
- CSV Dataset

## 📊 Key Business Questions Solved
1. Count of Movies vs TV Shows
2. Most common rating for each type
3. Top 5 countries by content volume
4. Longest movies on the platform
5. Content added in the last 5 years
... (list all 13)
